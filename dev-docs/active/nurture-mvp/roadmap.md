# Roadmap — G0 baseline closure

## Outcome

Deliver a reviewable Nurture MVP baseline whose production data boundary excludes My-Chat-owned workflow runtime persistence while retaining a separate, reproducible backend dev host.

## Milestones

1. Protection snapshot and clean formal branch.
2. UI/tooling retirement.
3. Cross-repository revision/hash pin.
4. Nurture production baseline.
5. Backend-private dev-host baseline.
6. Institution design documentation closure and G0 joint acceptance.

## Go/no-go rule

Each milestone must pass its targeted quality gate and have all discovered defects fixed before the next milestone starts.

## Final closure

G0 is complete only after PR merge, fresh-worktree verification, preserved-WIP coverage audit, and cleanup of temporary database/worktree/branch artifacts.
